# @svitejs/vite-plugin-template

PROJECT TAGLINE HERE

## Installation

```bash
pnpm i -D @svitejs/vite-plugin-template
```

## Usage

### add to vite config

```ts
// vite config
import { defineConfig } from 'vite';
import { pluginFunction } from '@svitejs/vite-plugin-template';

export default defineConfig({
	plugins: [
		pluginFunction({
			/* plugin options */
		})
	]
});
```
